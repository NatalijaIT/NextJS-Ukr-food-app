# 🇺🇦 Ukrainian Food App

A simple Next.js application showcasing the most popular Ukrainian dishes with recipes, descriptions, and images.

## 🚀 Pages

- Main page
- "Explore Meals" page with a list of meals
- "View Details" with details of the meal
- "Share page" with a form for a new meal

## 🛠️ Tech Stack

- Next.js
- React
- JavaScript
- CSS
- Node.js
- AWS S3 bucket for image storage

## 📦 Getting Started

### Prerequisites
Make sure you have installed:
- Node.js (v18 or later)
- npm or yarn

### Installation

```bash
git clone git@github.com:your-username/nextjs-ukr-food.git
cd nextjs-ukr-food
npm install
```

### How to run the project

1. Run this command in the terminal to create the DB
```bash
node initdb.js
```

2. Run it locally
```bash
npm run dev
```
